# SETS - STO Equipment and Trait Selector
A Star Trek Online build tool in Python

## Description

Pre-alpha version of build tool for STO

## Getting Started

### Dependencies

* Pillow
* requests-html

### Installing

> pip install -r requirements.txt

### Executing program

> python main.py
